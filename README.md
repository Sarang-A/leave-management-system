# Leave Management System

A web-based Leave Management System built with Django, allowing employees to apply for leaves and admins to manage approvals.

## Features

- Employee module — apply for leave requests
- Admin module — approve or reject leave applications
- CRUD operations using Django ORM
- Responsive UI with HTML, CSS, and Bootstrap
- Separate dashboards for employee and admin roles

## Tech Stack

- **Backend:** Python, Django
- **Frontend:** HTML, CSS, JavaScript, Bootstrap
- **Database:** SQLite
- **Version Control:** Git, GitHub

## Getting Started
```bash
git clone https://github.com/Sarang-A/leave-management-system.git
cd leave-management-system
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

Open browser → `http://127.0.0.1:8000`

## Usage

- **Employee:** Login → Apply for leave → View status
- **Admin:** Login → View all requests → Approve or Reject
